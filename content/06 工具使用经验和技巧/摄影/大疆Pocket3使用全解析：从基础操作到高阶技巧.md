# **注意事项**
- 全机身不防水、不防尘
- 开启后云台转动不受外力阻挡
- 不要固定在自行车、摩托车等剧烈震动的场景
- 工作温度0-40℃
- 每3个月充放电一次保持电池活性
- 长期不使用电池充到50%保存

## 一、产品核心功能概览
作为2023年底发布的革命性口袋云台相机，Pocket3凭借**1英寸传感器**、**旋转触控屏**和**三轴机械防抖**三大核心卖点，已成为vlog创作者的首选设备。其**179g超轻机身**支持4K/120fps视频录制，配合D-Log M色彩模式为后期调色提供充足空间。

- **实体按键**：

### 4. 云台模式
![[Pasted image 20250405205939.png]]
#### FPV云台模式
**FPV** = ​**F**irst ​**P**erson ​**V**iew（第一人称视角）

当云台处于FPV模式时：
- 三轴（俯仰/横滚/偏航）完全跟随手柄运动
- 无任何方向上的增稳补偿
- 模拟飞行器驾驶员头盔视角

$$ \text{自由度} = \begin{cases} 
俯仰轴 & \pm 90^\circ \\
横滚轴 & \pm 45^\circ \\
偏航轴 & 360^\circ \end{cases} $$

## 五、常见问题排查
1. **视频导出失败**：  
   - 检查DJI Mimo App是否为v4.8.2以上版本
   - 确保手机存储空间＞5GB

2. **云台校准异常**：  
   ```bash
   # 重置校准参数
   设置 → 云台校准 → 高级设置 → 恢复出厂校准
   ```

3. **低温环境拍摄**：  
   - 电池保温套（-10℃正常使用）
   - 避免骤冷骤热导致的镜头起雾

## 六、创作技巧与灵感
1. **旋转屏运镜**：开机时顺势旋转可获得电影级转场效果
2. **低角度模式**：倒置云台+延长杆模拟滑轨视角
3. **D-Log调色**：推荐使用LUT：
   ```css
   /* 电影感青橙色调 */
   shadows: RGB(25, 35, 45);
   midtones: RGB(120, 110, 100);
   highlights: RGB(200, 180, 160);
   ```
