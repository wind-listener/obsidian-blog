# 🏋️ 增肌减脂的经验化饮食配额  

## 📊 饮食配额表  
---

| 周期   | 性别 | 碳水            | 蛋白质         | 脂肪   |  
|--------|------|-----------------|----------------|--------|  
| ​**增肌期**​ | 男   | 3.5 - 4.5g      | 1.5 - 2g       | 1g     |  
|         | 女   | 3 - 3.5g        | 1.5g           | 1g     |  
| ​**减脂期**​ | 男   | 2.5 - 3g（初期） | 1.5g           | 0.8g   |  
|         | 女   | 2.5 - 3g（初期） | 1.2 - 1.5g     | 0.8g   |  
> 注：**单位**​：`g = 每kg体重的日摄入量（g/kg）`  
---

## 🔧 调整办法  
​**碳水作为热量变量**​：  
1. ​**增肌期**​：  
   - 试吃2周，体重能增约`1%`则不调整。  
   - 增长过快 → 减碳水（尺度`0.5g`）；增长过慢 → 增碳水。  
2. ​**减脂期**​：  
   - 试吃2周，体重能减约`2%`则不调整。  
   - 掉秤过慢 → 减碳水；掉秤过快 → 加碳水（尺度`0.5g`）。  

---

## ⚠️ 注意事项  
1. ​**体重变化后无需重算配额**​（增减仅几克），但减脂期体重降低`5-10kg`可重算一次。  
2. ​**休息日碳水需降低**​（比训练日减少`0.5g`）。  

---

## 🏥 大体重肥胖群体特殊调整  
| BMI    | 性别 | 碳水调整      | 蛋白质调整 | 脂肪调整 |  
|--------|------|--------------|------------|----------|  
| ​**>28**​ | 男   | 起始`2.5g`   | `0.6g`     | -        |  
|         | 女   | 起始`2.1g`   | `0.6g`     | -        |  
| ​**>32**​ | 男   | 起始`2.0g`   | -          | `0.5g`   |  
|         | 女   | 起始`1.0g`   | -          | `0.5g`   |  

---

### 📌 关键提示  
- ​**减脂期**​：从初期到末期碳水可递减，但需循序渐进！  
- ​**增肌期**​：优先通过碳水调节增重速度。
