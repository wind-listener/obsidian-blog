### 微分公式

微分是高等数学中研究函数变化率的核心工具。

#### 1. 基本求导公式
| 函数 | 导数 |
| :--- | :--- |
| $c$ (常数) | $0$ |
| $x^n$ | $nx^{n-1}$ |
| $e^x$ | $e^x$ |
| $a^x$ | $a^x \ln(a)$ |
| $\ln(x)$ | $\frac{1}{x}$ |
| $\log_a(x)$ | $\frac{1}{x \ln(a)}$ |
| $\sin(x)$ | $\cos(x)$ |
| $\cos(x)$ | $-\sin(x)$ |
| $\tan(x)$ | $\sec^2(x)$ |
| $\cot(x)$ | $-\csc^2(x)$ |
| $\arcsin(x)$ | $\frac{1}{\sqrt{1-x^2}}$ |
| $\arctan(x)$ | $\frac{1}{1+x^2}$ |

#### 2. 求导法则
* **常数与函数相乘法则：** $[cf(x)]' = cf'(x)$
* **和差法则：** $[f(x) \pm g(x)]' = f'(x) \pm g'(x)$
* **乘积法则：** $[f(x)g(x)]' = f'(x)g(x) + f(x)g'(x)$
* **商法则：** $\left[ \frac{f(x)}{g(x)} \right]' = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}$
* **链式法则：** $[f(g(x))]' = f'(g(x))g'(x)$，这是求解复合函数导数的关键。例如，求 $\sin(2x)$ 的导数，令 $f(u) = \sin(u)$, $u = g(x) = 2x$，则导数为 $\cos(2x) \cdot 2$。

### 积分公式

积分是微分的逆运算，用于求解曲线下面积、体积等。

#### 1. 基本不定积分公式
这些公式可以直接从上面的微分公式反推得到。

| 函数                       | 不定积分                      |
| :----------------------- | :------------------------ |
| $0$                      | $C$ (常数)                  |
| $x^n$ ($n \neq -1$)      | $\frac{x^{n+1}}{n+1} + C$ |
| $\frac{1}{x}$            | $\ln x+ C$                |
| $e^x$                    | $e^x + C$                 |
| $a^x$                    | $\frac{a^x}{\ln(a)} + C$  |
| $\cos(x)$                | $\sin(x) + C$             |
| $\sin(x)$                | $-\cos(x) + C$            |
| $\sec^2(x)$              | $\tan(x) + C$             |
| $\csc^2(x)$              | $-\cot(x) + C$            |
| $\frac{1}{\sqrt{1-x^2}}$ | $\arcsin(x) + C$          |
| $\frac{1}{1+x^2}$        | $\arctan(x) + C$          |

#### 2. 积分方法
当被积函数不是基本的类型时，需要使用一些技巧来求解。
* **换元法 (Substitution Method)：** 通过引入新的变量，将复杂的积分转化为简单的积分。
    * **例：** $\int 2x \cos(x^2) dx$。令 $u = x^2$，则 $du = 2x dx$，原式变为 $\int \cos(u) du = \sin(u) + C = \sin(x^2) + C$。
* **分部积分法 (Integration by Parts)：** 用于求解两个函数乘积的积分，其公式为：
    * $\int u dv = uv - \int v du$
    * **例：** $\int x \cos(x) dx$。令 $u=x$, $dv=\cos(x)dx$，则 $du=dx$, $v=\sin(x)$。
        * $\int x \cos(x) dx = x\sin(x) - \int \sin(x) dx = x\sin(x) + \cos(x) + C$。
* **分式积分法：** 当被积函数是有理分式时，通常通过**部分分式分解**来简化，然后逐项积分。

---
### 总结

* **微分公式**主要用于计算函数在某一点的斜率或变化率，是理解瞬时变化的基础。
* **积分公式**主要用于计算面积、体积或累积效应，是理解整体变化的基础。

这些公式是高等数学的基石，熟练掌握它们是解决更复杂问题的关键。
